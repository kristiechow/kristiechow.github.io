SAY HELLO
The Tiny Search Engine is composed of three parts, a crawler, an indexer, and a querier. Together, they crawl the web (starting at a given url), parse each found html page, store each word, its number of occurrences, and in which file it appears into an inverted-index structure. The user can then search for words and the Tiny Search Engine will return the files it appears in and how many times.

Developed in C, using Bash scripts for testing and a library of functions provided by Dartmouth CS50. Source code not yet available as it must be kept private for academic use.
