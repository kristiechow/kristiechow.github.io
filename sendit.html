SAY HELLO
Send It is an Android application developed in Android Studio that helps users make the most of their time by allowing them to schedule e-mails, text messages, and Facebook posts in advance. You'll never forget a birthday or business e-mail again!

Created using Java and XML, using Alarm services, Firebase databases, and Facebook API
